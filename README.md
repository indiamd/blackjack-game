# blackjack-game

This program simulates a game of blackjack. It was created after learning to play blackjack with real cards and in attempts to better understand it. 

main() function: prompts player to begin game, calls game function, prompts to end game, and keeps track of chips
game() function: simulates game of blackjack (deals hand, prompts player to draw new card or not, prints players cards + dealers top card)
other functions (printHand(), printDealer(), addCard(): all repeated functions used multiple times in game() for simplicity of game() function
